# Document Ingestion and Semantic Schema

이 문서는 기존 README의 Part 1 문서 적재 고도화 기록과 현재 적재 구조를 정리합니다.

## 적재 대상

- Excel (`.xlsx`)
- PDF
- HWP/HWPX
- 표 이미지

각 파일 파서는 원본 형식만 해석합니다. 추출한 표의 정제, 컬럼 의미 추론, 저장은 공통 파이프라인에서 처리합니다.

```text
파일
→ 형식별 파서
→ 병합 셀·헤더·행 복원
→ 공통 표 정제
→ 의미 스키마 생성
→ Parquet + ChromaDB + manifest
```

## E1. 이미지 처리와 병합 셀 복원

### 이미지 표

이미지 표 파서는 선과 셀 영역을 감지하고 OCR 결과를 원본 열 위치에 맞춥니다. 특정 문서의 고정 열 개수나 컬럼명을 요구하지 않습니다.

OCR 보정 정보는 품질 메타데이터로 분류하며 사용자 인물 카드에는 표시하지 않습니다.

### 병합 셀

Excel은 읽기 전에 실제 병합 범위를 확인합니다. 병합된 그룹 값은 근거가 있는 범위 안에서만 확장하며, 임의의 빈 셀 채우기로 다른 행에 값을 전파하지 않습니다.

## E2. 공통 스키마와 날짜 인식

원본 컬럼은 다음과 같은 의미로 분류합니다.

| 개념 | 예시 역할 |
|---|---|
| entity | 사람 이름, 기관 이름 |
| measure | 금액 |
| temporal | 날짜, 연도, 월, 연월 |
| category | 기수, 학과, 구분 |
| identifier | 발행번호, 학번, 문서번호 |
| description | 목적, 기준, 비고 |
| metadata/quality | 소스, OCR 품질 |

컬럼명만 보지 않고 실제 값 형식도 함께 확인합니다. 예를 들어 `학년`을 연도로 오인하지 않으며, `결제액(원)`과 `결제 일시` 같은 표현도 금액과 날짜로 인식합니다.

지원하는 날짜 구조:

- 완전한 날짜: `2026-01-03`
- 시간이 포함된 날짜: `2026-01-03 10:30:00`
- 연월: `2026-01`
- 분리 컬럼: `연도=2026`, `월=1`
- 월 전용 컬럼: 연도 질문에는 사용하지 않음

## E3. 원본 컬럼 보존

정제와 검색을 위해 파생 컬럼을 만들더라도 원본 컬럼을 삭제하거나 이름을 바꾸지 않습니다.

이 원칙은 다음 기능에 사용됩니다.

- 인물 카드에는 실제 원본 컬럼만 표시
- OCR confidence와 내부 식별자는 숨김
- 이름·금액 링크가 원본 행을 가리킴
- 다른 컬럼 구조에서도 런타임 의미 추론 가능

스키마 버전과 fingerprint를 저장해 같은 구조의 문서는 의미 매핑을 재사용합니다. 의미 추론 규칙이 바뀌면 스키마 버전을 올려 오래된 프로파일을 안전하게 갱신합니다.

## E4. 확장 계획

### Google Sheets

Google Sheets는 Excel과 같은 공통 표 파이프라인으로 연결할 수 있도록 원격 읽기와 인증 계층을 분리하는 것이 목표입니다.

### PDF

텍스트 PDF, 표 PDF, 스캔 PDF를 구분하고 표 추출 실패 시 OCR 경로를 선택하는 방향으로 발전시킵니다.

### 과도한 병합 구조

설명 행, 다단 헤더, 여러 표가 한 시트에 있는 문서를 독립 테이블로 나누는 검증을 계속 강화합니다.

## 주요 파일

| 파일 | 역할 |
|---|---|
| `ingest.py` | 파일 형식 판별과 적재 진입점 |
| `table_ingest_pipeline.py` | 공통 표 저장 파이프라인 |
| `table_parser.py` | 행 정제와 인물 파생 정보 |
| `semantic_schema.py` | 컬럼 의미, 단위, 개인정보 분류 |
| `parquet_store.py` | 표와 sidecar 저장 |
| `parsers/xlsx_parser.py` | Excel 시트 추출 |
| `parsers/pdf_parser.py` | PDF 본문·표 추출 |
| `parsers/hwp_parser.py` | HWP/HWPX 변환 |
| `parsers/image_table_ocr_parser.py` | 이미지 표 OCR |
